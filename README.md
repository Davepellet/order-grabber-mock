# Auto Order Grabber (Mock Project)

This is a SAFE mock simulation teaching how real-time order monitoring works.

## How it works
- `server.js` sends fake orders using WebSocket.
- `client.js` listens for orders and "auto-grabs" matching ones.
- No real platforms or APIs are used.

## Install
npm install

## Run server
node server.js

## Run bot
node client.js
